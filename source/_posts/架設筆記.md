## 前置準備
- 安裝 git
- 安裝 node.js

### 安裝 git
```bash
brew install git
```

### 安裝 node.js
``` bash
wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.33.2/install.sh | bash
```
重啟終端機
``` bash
nvm install stable
```

## 安裝 Hexo
``` bash
npm install -g hexo-cli
npm install hexo-deployer-git --save
```
